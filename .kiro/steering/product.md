# Product Overview

Headless Raspberry Pi lighting controller for studio environments. Enables network-based control of DMX lighting fixtures through QLC+ web interface.

## Core Purpose

Remote lighting control via browser/phone without physical access to the Pi. Anyone on the network can adjust lights through the QLC+ web UI.

## Key Components

- QLC+ open source lighting software (headless mode)
- ENTTEC DMX USB Pro interface (USB to DMX conversion)
- Raspberry Pi as dedicated controller host
- Web-based control interface (port 9999 by default)
- Optional landing page (nginx on port 80)

## Target Users

Studio operators, photographers, videographers, or anyone needing programmable DMX lighting control accessible from any device on the network.
